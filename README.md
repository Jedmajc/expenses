# Instrukcje dla aplikacji Expense Tracker

## Wymagania wstępne
- Android Studio
- JDK (Java Development Kit)
- Android SDK

## Klonowanie repozytorium
Aby sklonować repozytorium, użyj następującego polecenia:
```bash
git clone https://github.com/Jedmajc/expenses.git
```

## Otwieranie projektu
Otwórz Android Studio i wybierz "Open an existing project". Następnie wybierz folder z pobranym projektem.

## Synchronizacja z Gradle
Po otwarciu projektu, Android Studio poprosi o synchronizację z Gradle. Kliknij przycisk "Sync Now" na pasku powiadomień.

## Uruchamianie aplikacji
Aby uruchomić aplikację, wybierz konfigurację uruchamiania i kliknij zielony przycisk "Run" w Android Studio.

## Budowanie APK
Aby zbudować APK, użyj następującego polecenia w terminalu:
```bash
./gradlew assembleDebug
```
